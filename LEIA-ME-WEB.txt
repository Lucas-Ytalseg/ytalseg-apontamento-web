YTALSEG APONTAMENTO - VERSAO WEB COM LOGIN

O que e: app de apontamento na nuvem, com tela de login individual.
- Somente o admin cadastra/edita/remove usuarios (aba "Usuarios").
- Senhas guardadas com hash (seguras, nunca em texto puro).
- Os usuarios ficam no banco no Volume do Railway (nao somem em redeploy).

IMPORTANTE - precisa de Volume no Railway montado em /data
e a variavel DATA_DIR=/data (igual ao ERP).

Admin inicial: definido pela variavel ADMIN_USER / ADMIN_SENHA
(ou o padrao no codigo). Troque a senha pela aba Usuarios apos subir.

Arquivos:
  server.py          -> backend FastAPI (login + usuarios + serve o site)
  static/login.html  -> tela de login (porta de entrada)
  static/index.html  -> o app (protegido por login)
  static/script.js   -> logica do app + login + admin
  requirements.txt   -> fastapi, uvicorn
  railway.json/Procfile/nixpacks.toml -> deploy
